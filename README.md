# Housing In Padova

A modern 3D rental website for rooms, apartments, and houses in Padova.

This repository contains a full-stack application (frontend + backend) built with React, Vite, Tailwind CSS, Node.js, Express, and MongoDB.

Features
- Multi-language (English / Italian) UI
- Search, filters, saved listings, saved searches
- Messaging system between users and admin
- Admin panel to manage listings, users and messages
- Image upload (local storage)
- JWT authentication with role-based access

See backend/README.md and frontend/README.md for setup and run instructions.
